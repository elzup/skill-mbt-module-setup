# skill-mbt-module-setup

Claude Code skill for scaffolding a new MoonBit module project from [elzup/template-mbt-module](https://github.com/elzup/template-mbt-module).

## Install

```bash
claude skill add --from https://github.com/elzup/skill-mbt-module-setup
```

## Usage

In Claude Code, run:

```
/mbt-module-setup
```

The skill will interactively ask for module name, npm package name, GitHub repo, etc., then scaffold and customize the project.
